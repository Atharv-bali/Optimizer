## Runtime Optimization Analyzer (Compiler Design Project)

**Overview**

This project explores how different compiler optimization techniques affect the runtime performance of programs. The goal is to analyze, compare, and quantify how much optimization can improve execution speed.

It demonstrates the practical impact of compiler-level optimizations by running code under different optimization settings and measuring performance differences.

**Objectives**

Understand the role of compiler optimizations in improving performance
Compare execution time between non-optimized and optimized code
Analyze how different optimization levels affect runtime
Visualize performance improvements
**Key Concepts Covered**
Compiler Design Fundamentals
Optimization Techniques
Dead Code Elimination
Constant Folding
Loop Optimization
Inline Expansion
Time Complexity vs Runtime Performance
Benchmarking and Performance Analysis
